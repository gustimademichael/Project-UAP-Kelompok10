# ✨*GAME KECEMPLUNG*✨

## Deskripsi : 
Kecemplung adalah sebuah game sederhana di mana pemain mengendalikan sebuah keranjang untuk menangkap berbagai objek yang jatuh dari bagian atas layar. 
Pemain menggunakan tombol arah pada keyboard untuk menggerakkan keranjang ke kiri atau ke kanan. Setiap objek yang berhasil ditangkap akan menambah skor, 
sedangkan jika menangkap bom dapat mengurangi poin atau nyawa permainan. Seiring berjalannya waktu, kecepatan jatuh objek akan meningkat sehingga tingkat 
kesulitan permainan bertambah.
